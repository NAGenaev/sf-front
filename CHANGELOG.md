## [1.0.0] - 2026-06-21

### Added

- Full landing page UI: hero, featured studios, features, stats, testimonials, pricing, FAQ, contact, footer
- WebGL shader hero background (Three.js / React Three Fiber)
- Light and dark theme toggle with CSS variable design tokens
- English and Russian internationalization (`src/i18n/`)
- Scroll reveal animations (Framer Motion)
- Editorial hero typography (Cormorant Garamond)
- 80% UI scale via CSS (`--ui-scale`) with hero background scale compensation
- Vertically centered hero panel in the first viewport
- Open-source documentation (README EN/RU, CONTRIBUTING, CHANGELOG)
- Preview screenshots in `docs/`
- GitHub Actions CI workflow
