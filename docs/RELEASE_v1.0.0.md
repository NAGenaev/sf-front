## StudioFinder Landing UI — v1.0.0

Open-source landing page template for photo studio aggregators.  
MIT license — free for personal and commercial use.

**Русский:** open-source лендинг для агрегатора фотостудий. Лицензия MIT.

### Highlights

- WebGL shader hero background (Three.js)
- Light / dark theme toggle
- English and Russian i18n
- Framer Motion scroll animations
- Cormorant Garamond hero typography
- Built-in 80% UI scale with full-screen shader compensation
- Vertically centered hero panel

### Quick start

```bash
git clone https://github.com/NAGenaev/sf-front.git
cd sf-front
npm install
npm run dev
```

Open http://127.0.0.1:5173/
