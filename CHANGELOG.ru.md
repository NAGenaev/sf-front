# История изменений

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.1.0/) и [Semantic Versioning](https://semver.org/lang/ru/).

## [1.0.0] - 2026-06-21

### Добавлено

- Полный UI лендинга: hero, подборка студий, преимущества, статистика, отзывы, тарифы, FAQ, контакты, footer
- WebGL shader-фон hero (Three.js / React Three Fiber)
- Переключение светлой и тёмной темы через CSS-переменные
- Интернационализация EN / RU (`src/i18n/`)
- Анимации при скролле (Framer Motion)
- Editorial-типографика hero (Cormorant Garamond)
- Масштаб UI 80% через CSS (`--ui-scale`) с компенсацией для shader-фона
- Вертикальное центрирование hero-панели в первом экране
- Документация (README EN/RU, CONTRIBUTING, CHANGELOG)
- Скриншоты в `docs/`
- GitHub Actions CI
